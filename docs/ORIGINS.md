# Origens e decisões

- O SSHPLUS fornecido pelo usuário foi usado apenas como referência de funções e organização do menu.
- Nenhum módulo, binário ofuscado, chave, certificado ou instalador legado do SSHPLUS é distribuído na versão 1.4.
- O DTProto/DTProxy foi usado somente como referência visual e de experiência de gerenciamento; nenhum binário ou serviço deles é instalado.
- Nexus SSH_XHTTP é implementação própria, sem Xray, VLESS, VMess ou UUID de autenticação.
- Chisel é instalado do módulo oficial `github.com/jpillora/chisel`, fixado em `v1.11.5`.
- DNSTT é compilado do repositório oficial do projeto.
- BadVPN UDPGW é compilado do repositório oficial e configurado apenas em loopback.
